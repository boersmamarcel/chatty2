## Plan: Add Export Conversation Button

### Objective:
Enable users to export a conversation to a markdown file for further analysis. The export should include:
- The entire conversation content in markdown format.
- A user-friendly option to select the file location using the OS's file explorer.

---

### Tasks to Accomplish:

1. **Design UI**:
   - Add an **export button/icon** to the top-right corner of the `chat_view`.
   - Ensure the icon aligns with the existing design theme.

2. **Markdown Export Logic**:
   - Create a function to convert the chat history into markdown format.
     - Format messages with proper markdown conventions.
     - Include timestamps and sender info if required.

3. **Integrate File Explorer**:
   - Use the OS's native file selection dialog to provide a user-friendly way to choose the export location.
     - On Windows and macOS/Linux, use platform-specific APIs or libraries (like Node's `fs` module for Electron, or equivalent in Rust if you're implementing in Rust).

4. **Error Handling**:
   - Handle cases where the save process fails (e.g., permission issues, invalid locations).
   - Display friendly error messages to the user.

5. **Testing**:
   - Test the feature across supported operating systems to ensure smooth functionality.
   - Verify exported markdown follows correct formatting and includes all messages.

---

### Deliverables:
- A functional export conversation button in the chat UI.
- Exported markdown files with complete conversation histories.
- Error handling for save operations.
