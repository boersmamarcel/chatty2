1. Long-answer heuristic checks prompt length, attachments, newlines, and task verbs.
2. When `subtle_guidance_skeleton` is enabled and the 50/50 deterministic bucket admits the seed, empty assistant slots can reserve three ghost lines.
3. Widths are deterministic per prompt seed in the requested approximate 100/80/55% bands.
4. Lines live in one overflow-hidden reservation container.
5. First streamed text reuses the same assistant slot, so the skeleton is not rendered after content exists.
6. Reduced motion keeps static low-contrast lines with no shimmer.
